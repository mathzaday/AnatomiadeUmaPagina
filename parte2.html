Formulários Interativos: Coletando Dados

Informações Pessoais
Preferências

O formulario agora combina múltiplos elementos: <datalist> para autocompletar, <select> para uma lista fechada de opções

Para escolhas, usamos <input type="radio">

(Escolha unica) e <input type="checkbox"> (Escolha múltipla)

A tag <output> exibe dinamicamente o valor do <input type="range">, criando uma experiência interativa

Note tambem os atributos <placeholder>, que exibe um texto de ajuda no campo, e <required>, que torna seu prenchimento obrigatório.
